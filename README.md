# 🎬 Phoenix-Cinema System

> **Дипломный проект:** Разработка автоматизированной информационной системы для Кинопродюсерского центра «Феникс Синема».

## 🎓 О проекте
**Тема:** «Разработка информационной системы планирования и учёта киносъёмочного процесса».

Система предназначена для автоматизации работы администраторов и преподавателей киношколы. Она решает проблемы "двойного бронирования" техники, позволяет отслеживать состояние оборудования (исправно/сломано) и управлять съемочными группами.

---

## 🚀 Основные возможности

### 1. 🎥 Учет оборудования
* **Реестр техники:** Полная база камер, света, звука и реквизита.
* **Статусы состояния:** Возможность пометить оборудование как **«⚠ Сломано»** (с блокировкой бронирования) или **«✔ Исправно»**.
* **Умный поиск:** Живой поиск по названию и фильтрация по категориям.
* **Excel-интеграция:** Импорт новых позиций и экспорт текущего состояния склада в `.xlsx`.

### 2. 📅 Управление проектами и Бронирование
* **CRUD Проектов:** Создание, редактирование и удаление съемочных проектов.
* **Съемочная группа:** Назначение участников (учеников/сотрудников) в команду проекта.
* **Система бронирования:** Выдача техники под конкретный проект на определенные даты.
* **Защита от коллизий:** Алгоритм автоматически скрывает оборудование, которое уже занято другой группой в выбранную дату.

### 3. 👥 Ролевая модель доступа
Система разграничивает права для 4 типов пользователей:
* **Администратор:** Полный доступ, управление пользователями, просмотр логов.
* **Преподаватель:** Создание проектов, управление оборудованием.
* **Сотрудник:** Работа со складом, выдача техники.
* **Ученик:** Просмотр доступной техники, участие в проектах.

### 4. 🛡️ Безопасность и Аудит
* **Журнал событий:** Админ видит историю всех критических действий (кто удалил проект, кто изменил статус камеры и когда).
* **Валидация данных:** Защита форм от некорректного ввода.

## ⚙️ Установка и запуск (Windows)

Приложение готово к развертыванию на локальном сервере организации.

1.  **Клонируйте репозиторий** или скопируйте папку с проектом.
2.  **Откройте терминал** (CMD/PowerShell) в папке проекта.
3.  **Создайте виртуальное окружение:**
    ```bash
    python -m venv venv
    ```
4.  **Активируйте окружение:**
    ```bash
    venv\Scripts\activate
    ```
5.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
6.  **Запустите сервер:**
    * Вручную: `python app.py`
    * Или через скрипт: запустите файл `start.bat`
7.  **Откройте в браузере:** `http://127.0.0.1:5000`

> **Примечание:** При первом запуске база данных `cinema.db` создается автоматически. Первый зарегистрированный пользователь получает права Администратора.
